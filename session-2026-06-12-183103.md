# Session: 2026-06-12 18:31:03

## Summary
修复了 LUODA-Codex 的 Windows（NSIS）和 macOS（DMG）安装脚本，确保安装后桌面生成两个快捷方式：一个启动 Luoda-Codex（CODEX 启动器），一个启动 Luoda-Codex 管理工具。

## Lessons Learned
- PowerShell 的 here-string 和双引号字符串会展开变量，导致 bash/NSIS 脚本被破坏
- .NET 的 WriteAllBytes() 可以绕过 PowerShell 字符串解析
- 用 Python 脚本作为中间生成器避免 shell 语法干扰
- NSIS 使用 CreateShortcut 创建快捷方式

## Patterns
- Windows 用 NSIS + CreateShortcut，macOS 用 DMG + .app 捆绑
- 中文文件名在 NSIS 中需注意 UTF-8 编码

## Errors
- PowerShell 反复解析 bash 脚本中的语法
- 变量被 PowerShell here-string 展开导致内容损坏

## Preferences
- 安装后桌面需要两个清晰的快捷方式
- 优先用 Python + base64 绕过 PowerShell 编码问题
